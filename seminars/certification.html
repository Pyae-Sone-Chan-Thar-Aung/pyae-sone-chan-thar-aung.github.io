Reflection

Summer 2025 Training and Certification Summary

I am learning from the free online courses for the pre-training for the OJT based on the recommendations and the suggestions from the instructor Mr. Cris and Mr. Hizon.

1. Build a Computer Vision App with Cognitive Services
This course introduced the basics of computer vision and Microsoft's Cognitive Services. I learned how to integrate vision-based AI APIs into applications to detect objects, recognize faces, and analyze images with minimal coding.

2. Google Ads for Beginners
This certification provided foundational knowledge of how Google Ads works. I gained insights into setting up ad campaigns, managing budgets, selecting keywords, and tracking performance metrics to optimize online advertising.

3. Build a Free Website with WordPress
This hands-on training guided me through building, customizing, and managing a WordPress website. I developed skills in theme selection, plugin installation, page creation, and basic site optimization.

4. Introduction to Project Management with ClickUp
I learned the core principles of project management using the ClickUp platform. The course covered task creation, timelines, goals, and collaboration features for managing personal and team projects efficiently.

5. Getting Started with Microsoft Excel
This beginner-friendly course taught me essential Excel functions, including basic formulas, formatting, and spreadsheet navigation. It was a valuable foundation for working with data effectively.

6. Introduction to Data Analysis Using Microsoft Excel
Building on basic Excel skills, this course introduced data analysis techniques such as sorting, filtering, pivot tables, and charts. It emphasized interpreting and visualizing data for decision-making.

7. AI Awareness for Safety Professionals
I explored the role of AI in workplace safety, including risk prediction, incident analysis, and real-time monitoring. The course highlighted how AI tools can enhance proactive safety practices in industrial settings.

8. CSS Completion
This training covered Cascading Style Sheets (CSS), focusing on styling web pages through layout control, responsive design, and visual customization. It reinforced my ability to create attractive and user-friendly websites.

9. Raspberry Pi Completion
The Raspberry Pi course taught me how to set up and use a Raspberry Pi for hardware projects. I learned basic Linux commands, GPIO pin control, and programming with Python for IoT and automation applications.

Key Takeaways

  • Developed foundational skills in AI, data analysis, digital marketing, and web development.
  • Gained hands-on experience with tools like WordPress, ClickUp, and Microsoft Excel.
  • Enhanced technical abilities in computer vision and CSS for front-end development.
